* Emacs config

Emacs config built on [[https://github.com/SystemCrafters/crafted-emacs][Crafted Emacs]].

Clone this repo (with submodules):

#+begin_src shell
  git clone --recurse-submodules https://git.sr.ht/~mgmarlow/crafted-config ~/.emacs.d/
#+end_src

** Extras

- [[https://github.com/source-foundry/Hack][Hack font]]

** Update crafted-emacs

#+begin_src emacs-lisp
  git submodule update --init
#+end_src
